Tech Specs

Operating System

Pop!_OS 22.04 LTS, Ubuntu 22.04 LTS or Ubuntu 24.04 LTS

Processor

Intel® Core™ 3 100U: Up to 4.7GHz - 6 Cores - 2P+4E
Intel® Core™ Ultra 5 225H: Up to 4.9GHz - 14 Cores - 4P+8E+2LPE
Intel® Core™ Ultra 7 255H: Up to 5.1GHz - 16 Cores - 6P+8E+2LPE

Graphics

3 100U: Intel® Graphics
Ultra 5: Intel® Arc™ 130T GPU
Ultra 7: Intel® Arc™ 140T GPU
*Intel® Arc™ GPU requires using 2 x SO-DIMM

Memory

Up to 96 GB Dual Channel DDR5 5600 MHz

Storage

Up to 16 TB, 2 x M.2 PCIe Gen4 NVMe (I3), 1 x M.2 PCIe Gen5 NVMe and 1 x M.2 PCIe Gen4 NVMe (U5/U7), (System also supports 1 x M.2 (KEY M, 2242) with PCIe Gen4x4 for SSD) if the second 2280 M.2 Accessory is not installed.

Expansion

Front: 1 x USB 3.2 Gen2x2 Type-C, 2 x USB 3.2 Gen2 Type-A
Rear: 2 x HDMI 2.1 TMDS Compatible (4K@60Hz), 2 x Thunderbolt™ 4 ports (incl. DP 2.1 and USB4), 1 x USB 3.2 Gen2 Type-A,1 x USB 2.0 Type-A

Networking

Up to 2 x Intel 2.5 Gigabit LAN, 1 x Wi-Fi 7, Bluetooth 5.4

Video Ports

2 x HDMI 2.1 TMDS Compatible (4K@60Hz), 2 x Thunderbolt™ 4 ports (incl. DP 2.1)

Supply

Dependent on CPU:
Core 3 100U: 19V, 90W Power Adapter
Ultra 5 225H and Ultra 7 255H: 19V, 120W Power Adapter

Dimensions

117mm x 112mm x 54mm (4.63" x 4.41" x 2.13")
